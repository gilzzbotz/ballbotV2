## How to pakai?
```bash
> git clone https://github.com/bolaxd/ballbotV2
```
## Install dependency
Masuk ke Direktory dulu
```bash
> cd ballbotV2
```
```bash
> npm install
```
Or
```bash
> yarn install
```
